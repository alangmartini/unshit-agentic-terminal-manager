Profiling Dashboard

CPU and heap profiles for the Terminal Manager. Run the scripts, then hand the output files to the viewers below.

CPU Profile samply

Sample-based profiler output in Gecko/Firefox Profiler format.

  1. Run ./scripts/profile-cpu.ps1 (Windows) or ./scripts/profile-cpu.sh
  2. Exercise the app, then close it to stop recording
  3. Drop the output file on the zone below. The Firefox Profiler tab opens and loads it automatically via postMessage.

Memory Profile dhat

Heap allocation profile with per-allocation backtraces and live-bytes timelines.

  1. Run ./scripts/profile-memory.ps1 (Windows) or ./scripts/profile-memory.sh
  2. Exercise the app, then close it to flush the profile
  3. Drop the output file on the zone below, or click Open Viewer and load it there (DHat does not yet support automatic handoff)
